# Snake
2D Snake made with Bbop-Library

GameLoop :
- Modifier le framerate dans la GameLoop
Menu:
- plan du menu background 4-8 gros nuages s'ouvrent et se ferment en même temps que la transition In et Out
-> peut etre mis dans un TransitionManager

Settings :
- choix de la map (ciel, désert, forêt, mer/océan) -> change le snake / fruit
- choix de la difficulté (vitesse) + (taille++)
- son On Off
- musique On Off

Game Over:
- animation snake cogne ou mort
- son
-> fin anims -> retourn Menu

TexturePath :
- fichier texture path

Prologue :
- Ajout d'un mini prologue lors de la première connexion
- Rejoué le prologue dans les paramètres
